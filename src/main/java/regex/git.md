##  正则

###  1.匹配ip地址

> 要求：四段 每段范围 0-255 

* 分区间讨论

  * 0-99              [0-9]{1，2}

  * 100 -199      1[0-9]{2}

  * 200 -249      2[0-4] [0-9]

  * 250 -255      25[0-5]

    字符集 的范围 0-9 []不能 位数 {}拓展次

``` java 
(([0-9]{1,2}\.|1[0-9]{2}\.|2[0-4][0-9]\.|25[0-5]\.){3})([0-9]{1,2}|1[0-9]{2}|2[0-4][0-9]|25[0-5])
```
